# 🛒 ProShop - MERN E-commerce App

A full-stack e-commerce web application built using the MERN stack.

---

## 🚀 Features

- User Authentication (JWT)
- Admin Panel (Product, Order, User Management)
- Product Search & Pagination
- Price Filtering (Custom Implemented)
- Wishlist Functionality (LocalStorage)
- Shopping Cart & Checkout
- PayPal Integration (Sandbox)

---

## 🛠 Tech Stack

**Frontend:**
- React
- Redux Toolkit
- Bootstrap

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB
