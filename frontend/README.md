# 🛒 ProShop - MERN E-commerce Application

A full-stack e-commerce web application built using the MERN stack (MongoDB, Express, React, Node.js). This project demonstrates a complete online shopping experience with authentication, admin dashboard, product management, and custom features like wishlist and price filtering.

---

## 🚀 Features

### 👤 User Features

* User Registration & Login (JWT Authentication)
* Browse Products with Pagination
* Search Products
* Price Filtering (Min & Max Price)
* Add to Cart & Checkout
* Wishlist Functionality (LocalStorage-based)
* Order Placement & Order History

### 🛠 Admin Features

* Admin Dashboard
* Add / Edit / Delete Products
* Manage Users
* Manage Orders

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/vaibhavdevelopor/ProShop.git
cd ProShop
```

---

### 2️⃣ Install backend dependencies

```
npm install
```

---

### 3️⃣ Install frontend dependencies

```
cd frontend
npm install
```

---

### 4️⃣ Run the application

#### Start backend

```
npm run server
```

#### Start frontend

```
cd frontend
npm start
```
